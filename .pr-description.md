## Rule Behavior
The rule disallows the use of eval-like methods:
- `setTimeout` with string arguments
- `setInterval` with string arguments
- `setImmediate` with string arguments
- `execScript` with string arguments
- `Function` constructor

### Invalid Patterns
```typescript
setTimeout('alert("Hello")', 100);
setInterval('console.log("tick")', 1000);
const fn = new Function('a', 'b', 'return a + b');
```

### Valid Patterns
```typescript
setTimeout(() => alert("Hello"), 100);
setInterval(() => console.log("tick"), 1000);
const fn = (a, b) => a + b;
```
